# Brand Assets

This directory contains source and export-ready brand images used by the web app and platform metadata.

## Source Files

- `source-brand.png`: Original full-size source image used for derivative exports.
- `main_logo_dark.png`: Primary horizontal logo.
- `main_logo_dark_square_small.png`: Compact square logo.
- `watermark_logo.png`: Small watermark variant.
- `favicon.png`: Original favicon source.
